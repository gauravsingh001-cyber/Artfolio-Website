A frontend assessment project built using **React**, **TypeScript**, and **Vite**.  
It displays artwork data from the **Art Institute of Chicago API** with features like pagination, custom row selection, and responsive design using **PrimeReact** components.

## Features

- Fetch and display artwork data from a public API  
- Paginated table view with Next, Previous, and numbered navigation  
- Select or Deselect all rows on the current page  
- Custom row selection using an overlay input box  
- Responsive and modern UI design  
- TypeScript integration for strong typing and maintainability  

## Tech Stack

| Technology | Purpose |
|-------------|----------|
| React + Vite | Frontend framework and fast build tool |
| TypeScript | Type safety and structured development |
| PrimeReact | UI components like DataTable and OverlayPanel |
| Axios | API requests and data fetching |
| CSS | Custom styling and responsiveness |
